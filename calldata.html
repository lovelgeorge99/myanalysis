Call Data Analysis ( January 2019 )

Posted JUNE 26,2020

About

For this visualization project I have chosen to use the Call Data of the month of January 2019.
The dataset that I am using in this project is taken from Data.gov.in which was in the CSV form and was really unclean. Firstly I cleaned the data and removed all the irrelevant entries and then tried to analyse and output it in the forms of different graphs and plots using different python visualization libraries such as matplotlib and pyplot.

Cleaning of Data

The data which i downloaded from the Data.gov.in website had thousands of missing data and many NaN values.I dropped the data which was not neccessary using the pandas framework.The main problem with the data was that 50% of the State Name column was NaN but the latitude and longitude was provided.
Using the reverse_geocoder and the pprint libraries in python I wrote a function which took latitude and longitude of the columns having State Name as NaN as parameters and the function returned the location and I stored it in the corresponding State Name column.


Below u can see the raw or the Unlcean data that I downloaded.

We cannot find any mistakes or NaN values from just viewing the it. So i used the Pandas functions to find null values and other missing data


In the above output u can see the number of NaN values of each column. We can delete all the data of In Out Travelling having null values as it is a small number but we cannot do it for State Name.
So firstly i removed all the other NaN values except State Name and then replaced the NaN values of State Name with 0.

*After dealing with missing values I used pyhton's reverse_geocoder and pprint libraries to take latitude and longitude as input for a function and return the location and replaced it in the empty State Name.


Here you can see that once the function is execute the State Name has no NaN values which means our code run successfully.After that i saved the cleaned data set in a new CSV file cleanfile.csv

Analysing & Visualization of Data

Here comes the fun part where we can analyse the data which we just cleaned and visualize them using different graphs and charts.If you want the code for plotting the graphs you can download the full Juypter Notebook file given at the bottom of page.

Note : All graphs and charts are made using plotly which are interactive on real time.



The above piechart displays the percentage of contribution of different Operators. We can see that Reliance Jio has the maximum share of 49.2% and Airtel on the second place with 22.2% of the share.Vodafone,Idea and BSNL have a very small share as compared to Jio and Airtel



The above bar graph shows the exact number of users of different Operators in the month of January 2019. We can see that RJio has more than 25000 users and Airtel which is on the second place is not even having the half of users as RJio.



The above bar graph shows the comparison between the users of the top two Operators that is RJio and Airtel in different states.We can see that Maharashtra has the maximum numberof RJio users as compared to other states and Kashmir has the maximum number of Airtel users.



Now this graph is a Stack Graph showing us the Call Drop Category of different Operators.

Conclusion

We can easily conclude that Reliance Jio has the maximum numbers of users with satisfactory performance in most of the states in a short span of time as compared to Airtel or other users which have been providing services long before Reliance Jio.
